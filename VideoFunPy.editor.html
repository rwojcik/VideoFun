class VideoFunPy.editor.FrameEditorSmoothing(params)[source]¶

Smoothes frame either by gaussian blur or bilateral filter.

Parameters:params – comma separated string which specifies smoothing parameters.

First two values specifies smoothing range and last one specifies method (0 - gaussian blur, 1 - bilateral filter).

Example string ‘11,11,0’ will use gaussian blur with range of 5 pixels). Ranges are ceil’d to odd number.

frame_edit(frame)[source]¶

Performs smoothing.

Parameters:frame – Frame to be smoothed
Returns:Smoothed frame by chosen parameters in initializer.
